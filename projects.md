## 2. Azania School Portal
A robust school portal environment built to handle internal academic components and clean web routing.

*   **Tech Stack:** Node.js, Express.js (Backend Architecture), HTML5, JavaScript (Frontend UI), Custom Security Middlewares.
*   **Key Features:**
    *   Clean separation of frontend interface files from the core server logic (`server.js`).
    *   Secure route handling via backend middleware structures to protect user sessions.
    *   Custom error boundaries (`404.html`) to manage missing routes and enhance security configurations.
